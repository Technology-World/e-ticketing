# 🎟️ E-Ticketing App

A React-based event ticketing application where users can browse events, view event details, and go through a ticket purchasing flow.

## 🚀 Run locally

```bash
git clone https://github.com/Technology-World/e-ticketing.git
cd e-ticketing
npm install
npm run dev
```
